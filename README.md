# Unit Converter Web App

A simple web-based unit converter for:

- Temperature (°C ↔ °F)
- Distance (km ↔ miles)
- Weight (kg ↔ pounds)

## Tech Used

- HTML
- CSS
- JavaScript

## How It Works

1. Select a unit type
2. Enter a value
3. Click Convert — the result appears instantly!

## How to Run

- Clone or download the repo
- Open `index.html` in any browser
